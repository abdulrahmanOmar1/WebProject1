Summary

The project aims to develop a user-friendly web application for managing football teams specifically designed for the Birzeit University Faculty of Engineering and Technology. The application will allow users to register and login to their accounts, create teams, view team details, edit team information, and delete teams.

  • User Registration and Login
    • Users can create an account with a username, email, password, and confirm password.
    • Registration form validation ensures all required fields are filled correctly.
    • Users can log in using their email and password.
    • Upon successful login, users are redirected to the dashboard page.
    • Sessions are used to welcome the logged-in or registered user.
  • Dashboard Page
    • Displays a table with team information, including Team name, Skill level, Number of players, and Game Day.
    • Provides a button to create new teams.
    • Clicking on a team name redirects the user to the team details page.
  • Create New Team
    • Users can input the team name, skill level, and Game Day using a form.
    • Upon submitting the form, a new team is created and added to the dashboard table.
    • Users can easily return to the dashboard page by clicking on a dashboard link.
  • Team Details Page
    • Shows detailed information about a selected team, including Team name, Skill level, Game Day, and the names of players in the team.
    • Users can return to the dashboard page by clicking on a dashboard link.
    • Team creators have the option to edit the team information or delete the team.
    • Additional functionality allows users to add new players to the team using a simple form.
  • Edit Team Page
    • Displays a pre-filled form with the selected team's information that can be edited.
    • Users can modify the team's name, skill level, and Game Day within the form.
    • Submitting the form updates the team's information.
    • The page also provides a delete link to allow users to delete the team.
    • Clicking on the delete link removes the team and redirects the user to the dashboard.
