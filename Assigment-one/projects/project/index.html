Welcome to THE KING

The KING Football Team Manager is a web application developed for managing football teams specifically for the Birzeit University Faculty of Engineering and Technology. This user-friendly application allows users to register, login, create teams, view team details, edit team information, and delete teams.

Requirements:

  1. User Registration and Login:

    • The application has a registration page where users can create an account.
    • The registration form includes fields for username, email, password, and confirm password.
    • The registration form is validated to ensure all required fields are filled correctly.
    • Users can log in using their email and password.
    • Upon successful login, users will be redirected to the dashboard page.
    • Sessions are used to welcome the logged-in or registered user.

  2. Dashboard Page:

    • The dashboard page displays a table with four columns: Team name, Skill of the team (ranging from 1 to 5), the number of players in the team, and the Game Day for that team.
    • The dashboard page includes a button that allows users to create new teams.
    • Clicking on a team name in the table redirects the user to the team details page for that specific team.

  3. Create New Team:

    • The create new team page contains a form for users to input the team name, team skill level, and Game Day.
    • The new team is created and added to the dashboard table upon submitting the form.
    • After creating the team, users can return to the dashboard page by clicking on the dashboard link.

  4. Team Details Page:

    • The team details page displays information about the selected team, including team name, skill level, game day, and the names of players in the team.
    • Users can return to the dashboard page by clicking on the dashboard link.
    • Users who created the team have the option to edit the team information or delete the team.
    • When the user is in the Team Details page, the navigation includes an "Add Player" link. Users can add new players to the team using a simple form that includes an input field for player names and an add button.

  5. Edit Team Page:

    • The edit team page displays a pre-filled form with the selected team's information in an editable format.
    • Users can modify the team's name, skill level, and game day within the form.
    • After making the desired changes, users can submit the form to update the team's information.
    • The edit team page also provides a delete link to allow users to delete the team.
    • Clicking on the delete link removes the team and redirects the user to the dashboard.
